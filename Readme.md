# Snake & Ladder Game in JavaScript

A simple Snake & Ladder game simumation using Node.js and JavaScript. Dice roll is implemented using `Math.random()` function.

# Instructions
- Run `node snakeLadder.js`.

### Output
```stdout
// Player name | matches won
┌──────────┬────────┐
│ (index)  │ Values │
├──────────┼────────┤
│ Player 1 │ 501    │
│ Player 2 │ 498    │
└──────────┴────────┘
```
